# Evaluation Overview

Evaluation in healthcare AI must answer questions across multiple dimensions. Technical accuracy is necessary but not sufficient — a model can be accurate on average while performing unacceptably on specific subgroups; it can be technically functional while being unusable in practice.

## Prerequisites

| Prerequisite | Importance | Level |
|---|---|---|
| Completed development phase checklist | Necessary | — |
| Held-out test sets (not used during development) | Necessary | — |
| Understanding of statistical validation | Necessary | Intermediate |
| Access to representative users for usability testing | Helpful | — |
