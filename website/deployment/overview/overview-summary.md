# Deployment Phase Summary

Deployment should result in an AI system that is:

**Validated locally**: Performance confirmed on local patient data before going live

**Integrated**: Technical and workflow integration complete; disruption to existing workflows minimised

**Quality-controlled**: Mechanisms in place to detect data quality issues at inputs and outputs

**Monitored**: Performance monitoring dashboard operational; alert thresholds set; response procedures defined

**Logged**: All AI interactions recorded with sufficient detail for audit and investigation

**Trained for**: All intended users trained; training materials available; ongoing training planned

**Governed**: Roles and responsibilities assigned; accountability clear; incident reporting pathways established; periodic review scheduled

**Compliant**: Post-market obligations under applicable regulations fulfilled
