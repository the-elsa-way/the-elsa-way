# Development Overview

The development phase translates the design-phase decisions into a working AI system. The key activities are data management, model training, privacy implementation, human-AI interface design, and documentation.

## Prerequisites

| Prerequisite | Importance | Level |
|---|---|---|
| Completed design phase checklist | Necessary | — |
| Data access agreements in place | Necessary | — |
| Ethics approval obtained | Necessary | — |
| Understanding of machine learning | Necessary | Intermediate |
| Understanding of GDPR and data governance | Helpful | Beginner |
